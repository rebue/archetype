## 5. 创建虚拟网络

创建虚拟网络提供给docker容器互联使用，在 `Manager` 节点运行如下命令:

```sh
docker network create -d overlay --attachable rebue
```

## 6. 创建并运行常用容器

**注意:** 请使用文档中的 `Swarm` 安装方案，在 Docker Compose文件中加入如下内容，以便容器能互联互通

```yaml
networks:
  default:
    external: true
    name: rebue
```
